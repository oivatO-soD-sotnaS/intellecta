A fragmentação dos meios de comunicação no Instituto Federal do Paraná (IFPR) – Campus Foz do Iguaçu – tem se mostrado um problema recorrente. Isso ocorre devido à utilização de múltiplas plataformas pelos docentes para interagir com os alunos, o que pode gerar dificuldades no gerenciamento de contas, confusão entre disciplinas e dispersão da comunicação institucional. Visando unificar essas interações, o objetivo deste projeto foi desenvolver a aplicação web Intellecta, uma ferramenta de ensino digital. Para o delineamento do projeto, adotou-se uma abordagem de pesquisa qualitativa, utilizando formulários aplicados a professores e alunos da instituição, os quais obtiveram um total de 24 respostas. A partir da análise desses dados e da observação de plataformas consolidadas, como Google Classroom, ambientes virtuais de aprendizagem (AVA) e o SUAP, foi possível estabelecer o escopo inicial do sistema. Entre os requisitos mais desejados pelos usuários, destacam-se a integração com modelos de inteligência artificial e a possibilidade de importação de conteúdos entre disciplinas. Além disso, identificou-se como requisitos funcionais mínimos a existência de disciplinas, atividades avaliativas com submissão de textos ou arquivos, fóruns de discussão, entre outros. Atualmente, o projeto encontra-se na fase de implementação do código-fonte, com a modelagem do banco de dados concluída, protótipos de interface desenvolvidos, requisitos funcionais e não funcionais documentados, e os diagramas de entidade-relacionamento e de casos de uso finalizados. A implementação da arquitetura de microsserviços e o desenvolvimento do front-end e back-end encontram-se em estágio avançado, resultando em uma versão inicial funcional da plataforma.

> Este repositório permanecerá recebendo atualizações até o dia 15/12/2025.

## Configurando projeto
Para facilitar a configuração do projeto nós criamos um script em bash que automatiza todas as configurações necessárias. Todos os scripts de configuração se encontram na basta ***conf/***.
Dessa forma, para configurar o projeto, execute o comando abaixo:
```bash
./conf/main.sh
```
